二维轮询

同时轮询 Provider 和 Model。一个 Key 限流了换下一个 Key，一个 Model 限流了换下一个 Model，直到找到可用组合。

🔑

团队 Key 共享

多个成员各自购买套餐的 API Key 配置为不同 Provider，自动在所有 Key 之间轮询，叠加团队总配额。

⏱

自动恢复

被限流的 Provider / Model 进入冷却期，冷却结束后自动恢复为默认主配置（最强模型 + 主 Key）。

🌐

独立 Base URL

每个 Provider 可以配置不同的 API 地址，支持跨平台混合（阿里云、Anthropic、自建服务等）。

⚡

零依赖

纯 Node.js 内置模块实现，无需 npm install。一个 config.json + 一个 proxy.js 即可运行。

🔧

热更新

修改 config.json 后无需重启代理，自动检测配置文件变化并重新加载。也可通过 Web 界面编辑。
