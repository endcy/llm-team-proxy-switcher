Configuration

编辑 config.json 或查看各 CLI 工具的配置指南。

config.json

providers

Provider 配置数组。每个 Provider 包含 base-url（API 地址）、api-key（密钥）、models（可用模型列表）。按数组顺序轮询。

limiter-recovery-seconds

被限流的 Provider / Model 冷却恢复时间（秒）。默认 300（5 分钟）。到期后自动恢复为第一个 Provider 的第一个 Model。

port / bind

代理监听端口和地址。默认 9982 / 0.0.0.0（内网所有成员可访问）。如需限制本机使用改为 127.0.0.1。

maxRetries

每次请求最大轮询次数（跨 Provider + Model 组合）。默认 20。

requestTimeoutMs

上游 API 请求超时时间（毫秒）。默认 300000（5 分钟）。

Rotation Order

P0/m0 → P0/m1 → P1/m0 → P1/m1 → ...
依次尝试所有组合，429 时自动切换下一个。
